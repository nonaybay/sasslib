## How to use
**Assuming you are using the 7-1 architecture, place the repo inside the vendors folder:**
```
  sass
    │
    ├─ abstracts
    │
    ├─ base
    │
    ├─ components
    │
    ├─ layout
    │
    ├─ pages
    │
    ├─ themes
    │
    ├─ vendors
    │   │
    │   └─ sasslib
    |       │
    |       └─ pkg...
    │
    └─ main.scss
```
